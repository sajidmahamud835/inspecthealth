# Palette's Journal

## 2024-05-22 - Login Experience
**Learning:** Users lack feedback during authentication requests, which can lead to multiple clicks and frustration.
**Action:** Implementing loading states on submit buttons is a critical micro-interaction that should be standard across all forms.

## 2024-05-22 - Doctor Card Accessibility
**Learning:** Repeated "Book Appointment" links on a list page are ambiguous to screen reader users (e.g., hearing "Link, Book Appointment" 10 times).
**Action:** Always append the unique entity name to the action label (e.g., "Book appointment with Dr. Sarah").
